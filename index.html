What I do

Below is a quick overview of my main technical skill sets and technologies I use. Want to find out more about my experience? Check out my online resume and project portfolio.

HTML & CSS

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS provides the layout, for a variety of devices.

JavaScript

JavaScript is a dynamic computer programming language. It is lightweight and most commonly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages.

Jquery & Ajax

JQuery is a lightweight "write less, do more" JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website. AJAX is the art of exchanging data with a server, and update parts of a web page - without reloading the whole page.

Bootstrap 4

Bootstrap is a free and open source Front-End development framework for the creation of websites and web apps. The Bootstrap is built on HTML, CSS, and JavaScript (JS) to facilitate the development of responsive, mobile-first sites and apps.

Wordpress/Elementor

WordPress is a free and open-source content management system written in PHP and paired with a MySQL or MariaDB database. Features include a plugin architecture and a template system, referred to within WordPress as Themes.

Shopify

Shopify is a software service that offers you to create a website and use their shopping cart solution to sell, ship and manage products. Easy-to-use admin panel where you can add products, process orders, and enter store data.

React JS/React Native

React.js, a comprehensive JavaScript library for building user interfaces, has changed the way we think about Front-End development. React.js has grasped the interest of the open source community.

Visual Studio Code

Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, snippets, code refactoring, and embedded Git.

GitHub

GitHub, Inc. is a subsidiary of Microsoft which provides hosting for software development and version control using Git. It offers the distributed version control and source code management (SCM) functionality of Git, plus its own features.

Self QA

The current trend among software development teams is to move away from QA-based testing in favor of developer-based testing.

Adobe Photoshop

The world's best imaging and graphic design software is at the core of just about every creative project, from photo editing painting and compositing to digital painting, animation, and graphic design.

Bootstrap Studio

Bootstrap Studio is a proprietary web design and development application. It offers a large number of components for building responsive pages including headers, footers, galleries.

Zeplin

Zeplin focuses purely on improving the collaboration between designers and engineers. Just this March, 6,860,876 designs from Sketch, Figma, Adobe XD and Photoshop CC were exported to Zeplin by thousands of product teams.
